1) **`mogi up -> branch = <random_branch> && commit = <files_change>`**
2) **`mogi up -b ref -> branch = ref && commit = <files_change>`**
3) **`mogi up -b ref -m 'first commit' -> branch = ref && commit = 'first commit'`**
